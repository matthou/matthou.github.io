"""Part 2: Classification - Problem 1

    The general idea behind gradient descent is to minimize the loss function.  In order to do that,
    I first have to initialize the weights to zero.  Then, I have to calculate the current loss based on the 
    current weights and the target(y).  After calculating the loss, I need to find the steepest ascent in
    f(theta) which is the gradient.  Finally, I update the weights by subtracting the alpha times the gradient
    and repeat this process for T amount of time steps.
"""

#Training data (X, y), T time steps, alpha step
def grad_descent(X, y, T, alpha):
    m, n = X.shape #m = #examples, n = #features
    theta = np.zeros(n) #initialize parameters
    f = np.zeros(T)
    qf = []
    for i in range (T):
        qf.append(0)
    
    for i in range(T):
        #loss for current parameter vector theta
        qf[i] = 0.5*np.linalg.norm(X.dot(theta) - y)**2
        #compute steepest ascent at f(theta)
        g = X.T.dot(X.dot(theta) - y)
        #step down the gradient
        theta = theta - alpha*g
    return theta, f

def predict_g(info, coef):
    result = 0
    for i in range (len(info)):
        result += info[i]*coef[i]
    return result

Part 2: Classification - Problem 2

Loss Function $$ l(\theta) = \sum\limits_{i=1}^n y _{i}\log (p_{i}(\beta^{k})) + (1 - y _{i})\log(1 - p_{i}(\beta^{k})) $$

Take the derivative by doing chain rule:

$$ \frac{dl}{d\theta} = \sum\limits_{i=1}^n y _{i}(\frac{1}{p_{i}(\beta^{k})})(p_{i}(\beta^{k}))(1 - p_{i}(\beta^{k}))(x_{i}) + (1 - y_{i})(\frac{1}{1 - p_{i}(\beta^{k})})(-1)(p_{i}(\beta^{k}))(1 - p_{i}(\beta^{k}))(x_{i}) $$

Simplify the derivative:

$$ \frac{dl}{d\theta} = \sum\limits_{i=1}^n y_{i}(1 - p_{i}(\beta^{k}))(x_{i}) + (-1)(1 - y_{i})(p_{i}(\beta^{k}))(x_{i}) $$

$$ \frac{dl}{d\theta} = \sum\limits_{i=1}^n (y_{i} - y_{i}p_{i}(\beta^{k}) + (y_{i}p_{i}(\beta^{k}) - p_{i}(\beta^{k}))(x_{i}) $$

$$ B_{i} = \frac{dl}{d\theta} = \sum\limits_{i=1}^n (y_{i} - p_{i}(\beta^{k}))x_{i} $$

Plug this into the General Gradient Descent Equation:

$$ \beta^{k+1} = \beta^{k} + \alpha B_{i} $$

$$ \beta^{k+1} = \beta^{k} + \alpha\sum\limits_{i=1}^n (y_{i} - p_{i}(\beta^{k}))x_{i} $$

The gradient descent update equation for logistic regression that was given: $$ \beta^{k+1} = \beta^{k} + \alpha\sum\limits_{i=1}^n (y_{i} - p_{i}(\beta^{k}))x_{i} $$

'''Part 2: Classification - Problem 3

    In the logistic regression, I first have to initialize the weights to all zeros. For each step in the number
    of time steps(T), I calculate the predictions based on the sigmoid function and took the difference to get the
    loss.  Next, we can update the weights based on our alpha and the gradient which is calculated through the dot
    product of our loss and the training data.
'''
def sigmoid(z):
    return 1 / (1 + np.exp(-z))

def logistic_regression(X, y, T, alpha):
    
    theta = np.zeros(X.shape[1])
    
    for step in range(T):  
        predictions = sigmoid(np.dot(X, theta))
        loss = predictions - y
        gradient = np.dot(X.T, loss)
        theta -= alpha * gradient
        
    return theta
